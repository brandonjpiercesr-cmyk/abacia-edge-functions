# ABACIA Edge Functions

**80 Agents as Supabase Edge Functions**

## Quick Deploy

```bash
# Install Supabase CLI
npm install -g supabase

# Login to Supabase
supabase login

# Deploy all functions
npm run deploy:all
```

## Individual Deploy

```bash
supabase functions deploy sage --project-ref htlxjkbrstpwwtzsbyvb
supabase functions deploy dawn --project-ref htlxjkbrstpwwtzsbyvb
# etc...
```
